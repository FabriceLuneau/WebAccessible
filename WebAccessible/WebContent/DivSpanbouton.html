Div Span ... Button

If you use a tag like DIV, SPAN, P, ... and change apperancewith with CSS to design as button, you should:
  • Add aria property : role="button".
  • Add "" tabIndexproperty to set focus enabled with tab key.
  • manage the behavior with JavaScript code, set validation enable with space and enter key

Not to do

  • This button is not reachable with tab key.
  • It is not defined as button, and screen reader can not tell anything to user.
  • Only the mouse is manged by JavaScript.
