Switch buttons

Description

If you create a mutiple state button, you should :
  • Add aria property : role="button""".
  • Add "" tabIndexproperty to set focus enabled with tab key.
  • add aria property"aria-pressed" with values "true", "false", or "mixed"
  • manage the change ofaria property "aria-pressed" in JavaScript code

Example

Switch

Code

<span class="button1" role="button" tabIndex="0" aria-pressed="true" onClick="switchButton(this)" onKeyUp="if(isValidated(event)) {switchButton(this)}">
